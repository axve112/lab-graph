# lab-graph
作业
计算机图形学实验二：3D 坐标变换与 MVP 矩阵实现
一、实验目的

理解并实现 3D 空间中的模型变换、视图变换、投影变换。
掌握 MVP 矩阵的计算流程与坐标变换原理。
使用 Taichi 实现一个可旋转的 3D 三角形，并完成透视投影。

二、实验内容

定义一个 3D 三角形的顶点坐标。
实现模型变换矩阵（绕 Z 轴旋转）。
实现视图变换矩阵（将相机移动到世界原点）。
实现透视投影矩阵，完成从 3D 坐标到 2D 屏幕的映射。
通过键盘 A/D 键控制三角形旋转，实现动态观察效果。

三、核心原理

模型矩阵（Model）：对物体自身进行旋转、平移、缩放。
视图矩阵（View）：将世界坐标系转换到相机视角坐标系。
投影矩阵（Projection）：将 3D 坐标通过透视投影映射到标准设备坐标。
MVP 变换顺序：Pclip​=Mproj​×Mview​×Mmodel​×Plocal​

四、运行环境

Python 3.1
Taichi 图形库

安装依赖：plaintextpip install taichi

五、操作说明

按 A 键：三角形逆时针旋转
按 D 键：三角形顺时针旋转
按 ESC 键：退出程序
